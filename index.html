CSS to Figma Auto Layout Mapping

display: flex

Maps to Figma auto-layout frames with HORIZONTAL or VERTICAL direction based on flex-direction.

padding

CSS padding becomes Figma frame padding (top, right, bottom, left) with full shorthand support.

gap

CSS gap property maps directly to Figma's itemSpacing between auto-layout children.

justify & align

justify-content and align-items map to primaryAxisAlignItems and counterAxisAlignItems.

Sizing (Hug / Fill)

width: 100% becomes FILL. Auto or no width becomes HUG. Fixed pixel values stay FIXED.

Colors & Effects

Background colors, gradients, borders, border-radius, box-shadow and opacity are all preserved.

This is a Figma Plugin. Install it inside Figma to create real auto-layout frames on your canvas.

Run npm run build:plugin to build, then load the dist/ folder in Figma via Plugins > Development > Import plugin from manifest.
